Module Llvm_irreader

IR reader.

This interface provides an OCaml API for the LLVM assembly reader, the classes in the IRReader library.

exception Error of string
val parse_ir : Llvm.llcontext -> Llvm.llmemorybuffer -> Llvm.llmodule

parse_ir context mb parses the IR for a new module m from the memory buffer mb in the context context. Returns m if successful, or raises Error msg otherwise, where msg is a description of the error encountered. See the function llvm::ParseIR.
